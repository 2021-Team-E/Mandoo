## **how to start**

```bash
cd backend
```

가상환경 설정

```bash
python -m venv venv
cd venv
.\Scripts\activate
```

install

```bash
(venv) cd ../backend/
(venv) pip install -r requirements.txt
```

run

```bash
(venv) python app.py
```
